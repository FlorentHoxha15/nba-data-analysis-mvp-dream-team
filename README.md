# NBA MVP & Dream Team – Data Analysis (Excel)

This project analyzes NBA player performance using Excel to create an objective MVP evaluation and a balanced Dream Team selection.

## Project Goal
- Move beyond narratives and highlight performance using data
- Compare players across seasons using clear metrics
- Build a transparent MVP scoring framework

## Tools Used
- Microsoft Excel
- Pivot Tables
- Formulas
- Data Cleaning
- Data Storytelling

## Files
- `NBA_Data_Analysis_MVP_and_Dream_Team.xlsx` – full analysis and dashboards

- Data analytics

- Analytical thinking
